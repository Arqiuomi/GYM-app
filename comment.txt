In BD table 'user_train'

day_counter - подсчитывает число выполненных тренировок за месяц (в начале следующего цикла обнуляется)

positive_mark - вконце каждой тренировки пользователю предлагается оценить тренировку:
positive_mark{'трудно':0, 'круто':1, 'легко':2}
Эта оценка СУММИРУЕТСЯ с текущим значением positive_mark.
Вконце месяца сумма оценок определяет, было ли человек легко или трудно, нужно повышать рабочий вес/число повторов или нет.

weight_mult, number_mult - множители весов и колическтва повторений.
В таблице упражнений у каждого упражнения есть рабочий вес и количество повторений в рабочем подходе.
Мульты меняют эти значения в зависимости от пола, мышечного веса спортсмена, его уровня подготовки и прогресса.
Мульт равный 1 соответствует мужчине с мышечным весом 70 кг и начальным уровнем подготовки.




'statistic' - методы класса user

favourite exercise {'bench press':5}
Если упражнению ставится лайк, в словаре его значение увеличивается на 1 или добавляется в словарь со значением 1.
 Ключ с наибольшим значением публикуется по итогам месяца. Словарь ведётся в методе favourite_exercise класса user.

time
В этот список добавляется время каждой тренировки в минутах (секунды округляются)
time = [60, 90, 120, 76, 83]
Список ведётся в методе time класса user

hole time
Значение в колонке прибавляется время каждой проведённой тренировки за  месяц.
Значение вычисляется в методе hole_time класса user

max_weight
Максимальный вес среди всех упражнений.
Из метода 'plan' класса user берётся список упражнений. Из таблицы 'user_train' - weight_mult, из таблицы упражнений берётся вес
каждого упражнения из плана, домножается на мульт и, если значение больше существующего, заменяет в переменной кода
макс значение.
Значение вычисляется в методе max_weight класса user


Метод plan класса user
Составляет индивидуальный план пользователя на месяц. Принимает выбранные группы мышц, число дней тренировки,
выводит вложенные списки:
plan [day 1, day 2, day 4, day 5...]
day 1 = [Dip, Bench press, Hammer, Cross over] - названия должны совпадать с названиями упражнений в таблицы DB
